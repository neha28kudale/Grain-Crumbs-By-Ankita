# Grain Crumbs Website 🍫

A premium bakery website built for **Grain Crumbs by Ankita**, showcasing handcrafted brownies, brownie cakes, gifting solutions, and upcoming product collections.

## 🌟 Features

* Premium, mobile-friendly design
* Brownies & Brownie Cakes showcase
* WhatsApp order integration
* Gift Box enquiry workflow
* Bulk & Corporate order enquiries
* Delivery estimate calculator
* Founder story section
* Customer reviews
* Educational landing pages for upcoming collections
* Lead capture forms
* Responsive design for mobile, tablet, and desktop

## 🚀 Getting Started

### Clone Repository

```bash
git clone <repository-url>
cd grain-crumbs
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
