# Grain Crumbs 🍪

A premium bakery website built for Grain Crumbs, showcasing handcrafted cookies, brownies, millet-based treats, and gifting options.

## Features

- Product Catalog
- Product Details
- Order Enquiry Form
- Responsive Design
- WhatsApp Integration
- Admin Dashboard
- Google Search Console Ready
- SEO Optimized

## Tech Stack

- React
- Vite
- JavaScript
- CSS
- Vercel

## Live Website

https://graincrumbs.vercel.app

## Installation

```bash
git clone <repository-url>
cd grain-crumbs
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

Deployed on Vercel.
